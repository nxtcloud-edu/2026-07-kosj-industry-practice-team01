# 전입신고 상담 연결 기준

지식베이스에 명확한 근거가 없는 질문은 추측해서 답변하지 않는다.

외국인 배우자, 체류 자격, 가족관계, 세대 편입 가능 여부처럼 개별 판단이 필요한 질문은 담당 부서 연결로 안내한다.

답변 근거가 부족한 경우에는 거주 예정 행정동을 선택하게 하고 관할 주민센터 또는 민원행정 담당으로 연결한다.

담당자에게 문의할 때 사용자가 같은 설명을 반복하지 않도록 질문 내용을 요약한 문의 요약 카드를 제공한다.
