# 온라인 전입신고 신청 조건 및 세대합가 규칙

1. **온라인 신청 대상**
   - 본인(개인)이 새 주소지로 이사하는 경우 정부24를 통해 온라인 신청이 가능합니다.
   - 신청 시 간편인증 또는 공동/금융인증서를 통한 본인 확인이 필수입니다.

2. **온라인 신청 불가 대상 (방문 신청 필수)**
   - 미성년자가 단독으로 전입신고를 하는 경우
   - 기존 세대주가 포함된 곳으로 들어가는 '세대합가' 중 일부 예외 상황
   - 외국인이 포함된 세대의 전입신고 (외국인 등록 및 체류지 변경 신고 절차 필요)

3. **세대주 확인 절차**
   - 세대주가 아닌 세대원이 신청하거나, 다른 세대에 새롭게 들어가는 경우(세대합가)에는 전 세대주 또는 신 세대주의 확인(승인)이 완료되어야 최종 처리가 됩니다.
   - 세대주 확인은 정부24 로그인 후 [세대주 확인] 메뉴에서 승인할 수 있습니다.
